# Quick Start Guide

Get the Long Call Update Assistant running in 5 minutes!

## Step 1: Generate Icons (2 minutes)

1. Open `icons/generate-icons.html` in your browser
2. Icons appear automatically
3. Right-click each icon → "Save image as..."
4. Save in `icons/` folder as:
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`

## Step 2: Install Extension (1 minute)

### Chrome:
1. Go to `chrome://extensions/`
2. Turn ON "Developer mode"
3. Click "Load unpacked"
4. Select this folder

### Edge:
1. Go to `edge://extensions/`
2. Turn ON "Developer mode"
3. Click "Load unpacked"
4. Select this folder

## Step 3: Configure (1 minute)

1. Click extension icon in toolbar
2. Click "Configure Settings"
3. Enter:
   - Your SIEBEL ID
   - Your TEAM name
   - Your default SKILLSET (PS or STD)
4. Save

## Step 4: Test (1 minute)

1. In settings, enable "Test mode" (uses 1-minute timer)
2. Go to `https://vcc-na13.8x8.com`
3. Start a call (or simulate one)
4. Wait 1 minute
5. Notification should appear!
6. Click notification → Fill in ISSUE and REASON → Copy!

## Done!

Turn off test mode and you're ready to use it with real calls.
