ICON GENERATION INSTRUCTIONS
=============================

You need to create three icon files for the extension:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

OPTION 1: Use the HTML Generator (Easiest)
-------------------------------------------
1. Open the file "generate-icons.html" in your browser
2. The icons will be automatically generated
3. Right-click on each icon and select "Save image as..."
4. Save them in this directory as icon16.png, icon48.png, and icon128.png

OPTION 2: Create Your Own Icons
--------------------------------
Use any image editor (Photoshop, GIMP, Canva, etc.) to create:
- Icon theme: Clock/Timer
- Colors: Blue gradient (#4A90E2 to #357ABD)
- Suggested design: Clock face showing 22 minutes

OPTION 3: Use Online Generator
-------------------------------
Visit sites like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- https://icon-icons.com/icon/clock/

Create icons with the suggested design and download in the required sizes.

NOTE: The extension will NOT work without these icon files!
