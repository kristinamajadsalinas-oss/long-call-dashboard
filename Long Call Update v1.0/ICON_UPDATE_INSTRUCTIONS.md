# Icon Update Instructions

## How to Generate New TrendLife Icons

### Step 1: Open Icon Generator
Open this file in your web browser:
```
file:///home/kristinamajasa/Long Call Update v1.0/icons/generate-trendlife-icons.html
```

Or if web server is running:
```
http://localhost:8888/generate-trendlife-icons.html
```

### Step 2: Save Each Icon
You'll see 5 icons displayed:
- icon16.png (16×16)
- icon24.png (24×24)
- icon32.png (32×32)
- icon48.png (48×48)
- icon128.png (128×128)

**For each icon:**
1. Right-click the canvas
2. Select "Save image as..."
3. Save with the exact name shown (icon16.png, icon24.png, etc.)
4. Save to: `/home/kristinamajasa/Long Call Update v1.0/icons/`
5. **Overwrite** the existing old icons

### Step 3: Reload Extension
1. Go to `chrome://extensions`
2. Find "Long Call Update Assistant"
3. Click the reload button (circular arrow icon)

**The new TrendLife icons will now appear!**
